Subscribing to the Podcast

The quickest and easiest way to get sermons and other media from First Baptist Church is to subscribe to the podcast.

The easiest way to subscribe to a podcast is through iTunes. If you have a Windows computer, you will need to install iTunes. You can install iTunes by going to the iTunes Web site.

If you already have iTunes installed, you can click here to subscribe to the First Baptist podcast.

If you are using a different type of podcast client, the podcast feed is http://fbcmuncie.org/podcast/fbcmuncie.xml
